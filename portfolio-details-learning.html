Application web de formation

Ce projet consiste en un site de formation en ligne similaire à Udemy. La stack technologique utilisée comprend Angular pour le front-end, Node.js pour le back-end, et Prisma pour la gestion de la base de données. Les principales fonctionnalités incluent : • Système de login et d'inscription fonctionnel pour les utilisateurs. • Conception d'une landing page attrayante pour présenter les cours et les fonctionnalités du site. • Création d'une page de profil pour permettre aux utilisateurs de gérer leurs informations personnelles et leurs cours suivis.
